# Cook Flow (ALPHA!)

[![Flutter](https://img.shields.io/badge/Flutter-3.x-blue)]()
[![Platform](https://img.shields.io/badge/Platform-Android-green)]()
[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)

Приложение позволяет создавать собственную кулинарную базу, организовывать рецепты по категориям, составлять меню на несколько дней вперёд и автоматически формировать список необходимых продуктов.

CookFlow — приложение для хранения рецептов,
планирования питания и создания списков покупок.

## Возможности

* Создание и редактирование собственных рецептов
* Хранение ингредиентов и пошаговых инструкций
* Категоризация рецептов
* Поиск рецептов по названию
* Планирование меню
* Автоматическое формирование списка покупок
* Отметка купленных продуктов
* Работа на устройствах Android

## Установка

### Из исходного кода

```bash
git clone https://github.com/m4d0s/cookflow.git
cd cookflow
flutter pub get
flutter run
```

### Требования

* Flutter SDK 3.x или выше
* Android Studio или VS Code
* Android SDK
